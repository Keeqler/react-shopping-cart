<h1>🎉 Running</h1>

### ⚠️ Requirements

- [NodeJS](https://nodejs.org)
- [Yarn](https://yarnpkg.com)

### 🔥 Running the React app

Install the dependencies:

```shell
$ yarn
```

Start the React app:

```shell
$ yarn start
```

You will be able to access it in your browser at http://localhost:3000
